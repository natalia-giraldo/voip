
# 📍Projet VoIP Asterisk 22 📞
Titre : Voice over Internet Protocol
Auteur : Evan Bonnal, Natalia Giraldo, Alexandre Berdejo, Karim Chanfi
Formation : Bachelor IT en Cyber
Période : 16 février 2026 – 20 février 2026 (1 semaine)
Établissement : La plateforme_
______________________________________________

Ce dépôt contient les configurations et scripts pour le déploiement d'un serveur VoIP complet sous Debian 12 avec Asterisk 22 (compilé depuis les sources).

## 🚀 Fonctionnalités
Appels Audio : Configuration PJSIP (Alice & Bob).
Serveur Vocal Interactif (SVI) : Menu dynamique ("Tapez 1 pour Alice...").
Synthèse Vocale (TTS) : Intégration hybride Google TTS.
Messagerie Vocale : Boîte vocale avec protection PIN.
Automatisation : Script de prospection téléphonique automatique (Call Files).
Gestion Horaire : Mode "Ne pas déranger" (DnD) automatique 9h-18h.
Supervision : Monitoring via CLI et sngrep.
