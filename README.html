VoIP with Linphone - Documentation

Getting Started

Welcome to VoIP with Linphone, a comprehensive solution for voice calls throug internet. This documentation will guide you through installation, configuration, and usage.

Installation

git clone https://github.com/natalia-giraldo/voip.git
cd voip
# Here goes the installation steps

1st step

Pre-rquirements

Pour compiler un programme , on va avoir besoin de quelques paquets et avoir son système à jour :
sudo apt update && sudo apt -y upgrade

# Basic tools

sudo apt -y install build-essential git wget curl subversion automake autoconf libtool \ pkg-config unzip cmake

# Asterisk dependencies

sudo apt -y install libedit-dev uuid-dev libxml2-dev libssl-dev libncurses5-dev libsqlite3-dev \ libjansson-dev libxslt1-dev libsqlite3-dev libiksemel-dev libspeexdsp-dev \ liblua5.2-dev libcurl4-openssl-dev libpopt-dev libpq-dev unixodbc-dev

# Network tools/debug (Optional)

sudo apt -y install sngrep tcpdump htop ufw fail2ban

Configuration

Configure the project by modifying the following files:

  • config/config.json - Main configuration file
  • settings.ini - Additional settings

2nd step

Asterisk is no longer distributed in a debian package, we will need to retrieve the sources. We will take the current LTS (20.x) from the official website:

cd /usr/src sudo wget https://downloads.asterisk.org/pub/telephony/asterisk/asterisk-22-current.tar.gz sudo tar xvf asterisk-22-current.* cd asterisk-22.*/

You can download the latest version from the official website: https://downloads.asterisk.org/pub/telephony/asterisk/

3rd step

Unzip the folder from the CL with the command: tar xvf nomdufichier.tar.gz

4th step

Sound & script retrieval

Asterisk provides a script to prepare the environment: sudo contrib/scripts/install_prereq install sudo contrib/scripts/get_mp3_source.sh

5th step

Compilation & Startup

sudo ./configure

You'll see the Asterisk logo once the installation is complete

Usage

# Example usage
./run.sh --option1 --option2

For more detailed usage, see the examples directory.
